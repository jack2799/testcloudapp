# TestCloudApp

## Overview
This is a Flutter test application designed to demonstrate various features of the Flutter framework.

## Features
- Cross-platform compatibility
- Responsive UI
- Widgets showcasing Flutter capabilities
- Test automation

## Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/jack2799/testcloudapp.git
   ```
2. Navigate to the project directory:
   ```bash
   cd testcloudapp
   ```
3. Install dependencies:
   ```bash
   flutter pub get
   ```
4. Run the app:
   ```bash
   flutter run
   ```
